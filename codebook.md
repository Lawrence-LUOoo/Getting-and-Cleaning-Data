# Codebook

The raw datasets are download from [here]("https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip")

The raw datasets are combined into a single dataset, then cleaned up into tidy dataset.

## Variables in the `tidy_dataset.txt` are described below:

- `subjectId`

	Subject identifier, integer, ranges from 1 to 30.

- `activityId`

	Activity identifier, string with 6 possible values: 
	- `WALKING`: subject was walking
	- `WALKING_UPSTAIRS`: subject was walking upstairs
	- `WALKING_DOWNSTAIRS`: subject was walking downstairs
	- `SITTING`: subject was sitting
	- `STANDING`: subject was standing
	- `LAYING`: subject was laying

## Transformations

The following steps were applied to the source data:

1. Merges the training and the test sets to create one data set.
1. Extracts only the measurements on the `mean` and `standard deviation` for each measurement.
1. The `activityId` (originally coded as integers between 1 and 6) were replaced with descriptive activity factor names.
1. The variable names were replaced with descriptive variable names using `gsub`
1. From the data set in step 4, the final data set was created with the average of each variable for each activity and each subject.
